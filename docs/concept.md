# Flipper OS Concept (from September 2025 presentation)

Source: cdn.flipper.net/flipper_os_first_concept_september2025.pdf

## Core Ideas
1. Profile-based system: built-in profiles (WiFi Router, Wayland Desktop) + user profiles
2. Boot menu for profile selection with info (size, dates)
3. Built-in profiles: can Reset (to default) and Clone, CANNOT delete
4. User profiles: can Clone and Delete, CANNOT reset
5. New profile: boots minimal system with random name
6. Atomic updates: system update from any running profile, only needs internet
7. Package types: classic deb, complex containerized (Flatpak?)
8. External storage: SD card for profiles + user data
9. Running system shows: current profile name, size, created/modified dates
10. Clone/rename, delete/reset operations available from running profile
